## [0.0.1]

### Added

- Initial release of `flutter_anam_sdk`
- Real-time avatar streaming via WebRTC
- Two-way audio communication with microphone controls
- Text messaging interface
- Event-driven architecture using `EventEmitter`
- Support for custom personas
- Cross-platform support (iOS, Android, macOS, Web)
- `disableClientAudio` option to disable client microphone access and set audio transceiver to receive-only mode
- Support for custom transcription and turn-taking logic
- WebSocket proxy support for server-side session management
